
This repository contains a comprehensive data analysis project commissioned by the executives  of BrightTV. The primary objective of this study is to leverage viewership analytics to drive subscriber growth and optimise the Customer Value Management (CVM) strategy for the current fiscal year.
By synthesising user profiles and transactional session data, this analysis identifies the core drivers of consumption and provides a strategic roadmap for increasing platform engagement.


Project Scope & Deliverables

The analysis addresses four critical pillars required for the CVM team's strategic planning:

User & Usage Longitudinal Trends: A deep dive into behavioral patterns and demographic segmentation.

Consumption Determinants: Statistical identification of variables (temporal, content-based, or technical) that drive high viewership.

Tactical Programming Recommendations: Data-backed content strategies to mitigate "trough" periods in the weekly consumption cycle.

Subscriber Acquisition & Retention Initiatives: High-level strategic proposals to scale the BrightTV ecosystem.


 Methodology & Technical Constraints

To ensure data integrity and local relevance, the following technical requirements were implemented:
All transaction data was converted from UTC to South African Standard Time (SAST) (p. 1).

Granular Session Analysis: Consumption was modeled at the individual session level to capture precise user engagement metrics (p. 1).

Feature Engineering: Analysis of user profiles merged with viewership transactions to identify correlation between user segments and content preferences.
